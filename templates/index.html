Agenda Highlights

Opening Keynote: India’s Tech Revolution

Exploring how AI, Cloud, and emerging technologies are transforming India’s digital future and shaping tomorrow’s innovators.

Dive into AI, Web3, Cybersecurity, and Cloud with live demos, real-world case studies, and speaker-led hands-on sessions.

Workshop: From Code to Cloud

Hands-on with AWS - build, deploy, and scale serverless applications using Lambda, API Gateway, and more.
